# Cyberware

> All prices are in Creds

Each **Cyberware** installed reduces **Max CTR** by 1. You can install multiple of the same cyberware, each costing 1 Max CTR to install.

**Trigger** a bonus effect by taking stress equal to the trigger die. If duration is unclear, roll the trigger die to determine number of rounds an effect lasts.
